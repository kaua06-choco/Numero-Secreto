## 📖 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais de desenvolvimento Front-end, como manipulação do DOM, lógica de programação e interação com o usuário utilizando JavaScript puro.

A aplicação conta com uma interface simples, responsiva e intuitiva, além da utilização de síntese de voz para tornar a experiência mais dinâmica.

---

## ✨ Funcionalidades

- 🎲 Geração aleatória do número secreto.
- 🔢 Escolha de números entre **1 e 10**.
- 💡 Dicas indicando se o número secreto é maior ou menor.
- 📊 Contador de tentativas.
- 🔄 Reinício da partida sem recarregar a página.
- 🔊 Leitura das mensagens utilizando ResponsiveVoice.

## 🚀 Como Executar

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/jogo-numero-secreto.git
```

2. Acesse a pasta do projeto:

```bash
cd jogo-numero-secreto
```

3. Abra o arquivo **index.html** em seu navegador ou utilize a extensão **Live Server** no Visual Studio Code.

---

## 🎮 Como Jogar

1. Digite um número entre **1 e 100**.
2. Clique em **Chutar**.
3. O sistema informará se o número secreto é maior ou menor.
4. Continue até acertar o número.
